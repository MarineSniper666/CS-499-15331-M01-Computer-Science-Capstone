About This Portfolio

This ePortfolio showcases my work in the CS 499 Computer Science Capstone. The primary artifact featured in this portfolio is an Inventory Management Mobile Application developed in CS 360: Mobile Architecture and Programming. The project demonstrates skills in software design and engineering, algorithms and data structures, and database integration through planned enhancements and analysis.

Artifact Overview

Artifact: Inventory Management Mobile Application
Course: CS 360: Mobile Architecture and Programming
Platform: Android
Database: SQLite
Primary Features: Inventory tracking, login system, SMS alerts
Focus Areas: Software Design, Algorithms, Databases

Planned Enhancements

  • Inventory analytics calculations for cost, value, and profitability review
  • Profit and loss monitoring with SMS alerts when items enter a loss condition
  • Inventory transaction tracking through a relational transaction history table
